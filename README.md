# 푸드메이트\(FoodMate\) 공식 매뉴얼

## 푸드메이트\(FoodMate\)

푸드메이트\(FoodMate\)는 2020년 10월에 공개한 시각장애인을 위해 제작한 음식 음성안내 IoT 서비스입니다. 숟가락 또는 젓가락으로 음식을 가리키고 버튼을 누르면 인공지능으로 식판에  어떤 음식이 담겨있는지 판단하고 음성으로 알려주는 기능을합니다.

하단링크를 클릭하거나 좌측의 메뉴에서 원하는 도움말을 찾아보세요.
